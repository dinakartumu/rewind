## `get_schema`

Return the annotated Rewind database schema: every queryable table with its columns, types, semantic notes, join keys, and global conventions (single user_id, ISO timestamps, rating scales, image URL composition).

This tool takes no parameters.

## `query_rewind`

Run a read-only SQL SELECT against the Rewind SQLite database.

<ParamField path="sql" type="string" required>
A single read-only SELECT (or WITH … SELECT) statement. Do not include a trailing semicolon. A LIMIT is applied automatically. Call get_schema first for table and column names.
</ParamField>
